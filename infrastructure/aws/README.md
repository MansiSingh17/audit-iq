# AWS Infrastructure

AWS configuration files for AuditIQ deployment.

## Files
- `ecs-task-definition.json` - ECS Fargate configuration
- `s3-bucket-policy.json` - S3 bucket access policies
- `iam-policy.json` - IAM role policies
- `cloudformation-template.yaml` - Infrastructure as Code
- `deploy.sh` - Deployment script

## Usage
```bash
./deploy.sh
```
